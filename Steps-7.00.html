Loading Data 7

After you download the attached files to your computer, open a new file in Power BI Desktop

  1. Click Get data on Power BI Desktop
  2. Under Get Data select Text/CSV
  3. Select Electric car sales by Model in USA CSV file and open it
  4. Click on the load button to load the data

Repeat same step to load the Excel file:

  1. Click Get data on Power BI Desktop
  2. Under Get Data select Excel
  3. Select Model EV Type Excel file and open it
  4. Click on the load button to load the data


Always make sure you load the correct sheet(s) when loading an Excel workbook and do not load unnecessary data!
