Transform the data 8

First Table:

In the Data area, Select the first table and Click Transform Data button in the desktop ribbon:


rename your table: “US EF Car Sales” (because the original title does not match the data we have in the table, I have decided to put a title that describes the data correctly) using the Name space to the right of the window.

Select Transform tab from the ribbon, then click on Use First Row as Headers. It will promote the first row to column names.

Make sure to check for empty and null fields, and replace them with 0 using Replace Values button in the ribbon.

After:



now we need to unpivot the dated columns which will allow us to have a new dimension: Date.

Select the First 3 columns, and right click on any of the selected headers. In the drop-down menu select “Unpivot other Columns”


Rename the new columns: Attribute as Date and Value as Quantity Sold

now we need to change the date column to date data type:


Replace the Delimiter - with “, 20”, then change data type to Date

 


Note: Sometimes, after you unpivot the columns above, you may get the year 2 digits before the short Month name. I have not found an explanation for it. You will have to follow different steps:

Select the attributes column, Split it by Delimiter.

You will have attributes 1 and attributes 2.

Now Select Attributes 2 then Attributes 1 ( it is very important to follow the order)

Merge the 2 columns using delimiter – and name the merged Column Date

Replace the Delimiter - with “, 20”, then change data type to Date

You will get same results as shown above in the screenshots


Save and Apply
