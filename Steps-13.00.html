Visuals for KPIs 13

Adding Visual card for Qty Sold, Sales Per month, Qty MoM, and Qty YoY

Steps

Qty Sold Card Visual

Open Visualization Pane

Click on Card Visual, it will open in the dashboard

Drag TotalSales  Measure into the fields.


Go to formatting pane:

General:

Size Height = 90

Size Width = 192

Position H = 0

Position V = 112

Turn Title on, change it to Qty Sold

Change Font to Bold

Change Text Color to white #FFFFFF

Change Background to black #000000

Change Title alignment to Center

Turn Background on: Color #FFFFFF Transparency 40%

Turn Visual border on: #cccccc

Callout Value: Color  #a38600,  Value Decimal 2, Font Size 25 Bold

Category label OFF


 

 

 

Qty Per Month Card Visual 

Copy and paste the previous visual and make the following modifications

Drag Qty Per Month  Measure into the fields.

Go to formatting pane:

General:

Size Height = 90

Size Width = 192

Position H = 0

Position V = 201

Change title to Qty Per Mth

Callout Value: Color  #0D6ABF


  

Qty MoM Card Visual 

Copy and paste the previous visual and make the following modifications

Drag Qty MoM  Measure into the fields.

Go to formatting pane:

General:

Size Height = 90

Size Width = 192

Position H = 0

Position V = 291

Turn Title on, change it to Qty MoM

Turn Visual border on: #cccccc

Callout Value: Color  #E66C37

Go to filters pane:

Drag Year from Calendar to filters on this visual

Change Filter type to basic filtering

Filter out the first year in the list (2012)
