Calendar Table and Data Model 9

Create a Calendar table Using DAX:

In the report view, select modeling tab from the ribbon
click new table

Enter and save the following Dax formula in the formula bar, and save:


Calendar = ADDCOLUMNS (

 

CALENDAR (Min('US EF Car Sales'[Date]), MAX('US EF Car Sales'[Date])),

 

"Year", YEAR ([Date] ),

 

"Monthnumber", FORMAT ([Date], "MM" ),

 

"MonthNameLong", FORMAT ([Date], "mmmm" ),

 

"Quarter", "Q" & FORMAT ([Date], "Q" )

)

 

 

Creating Relationships between the tables


Go to data view,

click and drag date from calendar table to US EF Car Sales date
click and drag Model from EV Type table to US EF Car Sales model
