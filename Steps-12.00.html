Dashboard background, Logo and Title 12

Adding dashboard background

Click on an empty space on the work space, in the visuals pane, select formatting button

Select Canvas Background

Click browse and select the picture attached

Change transparency to 50%

 

 Adding the logo to the dashboard

Select Insert tab in the desktop ribbon, then click image

Select the attached logo image

in the format image pane, select general

Size Height = 112

Size Width = 192

Position H = 0

Position V = 0

Turn background on

Adding Dashboard Title

Select Insert tab in the desktop ribbon, then click Text Box

Top Line: “US Eco-Friendly Car Sales”
Font Courier New size 24 Bold Color #0D6ABF Centered

Bottom Line “2012 – 2019”
Font Courier New size 18 Bold Color #0D6ABF Centered

in the format text box pane, change

Size Height = 112

Size Width = 900

Position H = 191

Position V = 0


Adding the Colaberry Logo

Select Insert tab in the desktop ribbon, then click image

Select the attached logo image

in the format image pane, select general

Size Height = 42

Size Width = 191
