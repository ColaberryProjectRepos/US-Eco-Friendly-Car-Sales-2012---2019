Slicer and Visuals 14

Create the Year Slicer

Copy and paste the Qty Sold card visual

Remove the field data

Change the visual type to slicer by clicking on slicer visual

Drag Year from Calendar to the Field

Go to formatting pane, select general

Size Height = 112

Size Width = 190

Position H = 1090

Position V = 0

Change background transparency to 0

Turn off visual border

Change Title to YEAR

Select visual

Turn off slicer header

Values: Color #000000 Size 9 Bold



Create Monthly Sales Chart:

Click Line and Clustered Columns Chart in visuals pane

Add Month then year from calendar to the shared axis field

Add Qty Per Month to column values

Add Qty MoM and Qty YoY to line values

Add TotalSales to Tooltips field, Click the drop down arow and select show value as % of GT

Add TotalSales to Tooltips field


Go to formatting pane:

General:

Size Height = 359

Size Width = 1088

Position H = 192

Position V = 112

Change Title to Monthly Sales

Size 14 Bold color white #FFFFF background #000000 Center


Effects:

Background white Transparency 40%

Turn Visual border on: #cccccc


Visual:

Select Y axis, turn off title

Turn on Secondary y axis

Go to range and turn on align zeros

Select legend, change line style to lines and markers

Select Lines, go to colors
Qty MoM Color  #E66C37
Qty YoY color #66839F

Turn on Shade Area

Turn on Markers

Select Columns change Color to #0D6ABF

Turn on Data labels for columns, Position Inside End
